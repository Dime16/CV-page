About This Project

Where-Too app is an idea that i had when I started learning Node.js. This app lets you pick a town anywhere in the world, and depending on the weather lets you pick a place to go. If the weather is sunny it gives you Parks, Zoo, Amusment Parks and if the weather is rainy it gives you options like Movie Theaters, Shopping Mall, or Casino. Cafe and Bar are standard in any search. Allso it gives the Night Club option if it's after 8 PM. The app i build in Node.js with AXios as main module. It gets data from 3 API's, first it gets the geolocation from the place searched, then it gets the weather for that loaction from Dark-Skies API and then it gets the info from Google API for the places required. The app sorts the places by rating and then it shuffles them to get some random look when you search the same city twice or more times.
